# Getting started with SMS Gateway

Welcome to `sms-gateway`. This guide gets you from zero to a running local copy.

## Prerequisites

- Docker
- Access to the adaptive-labs GitHub org

## Run locally

```bash
git clone git@github.com:adaptive-labs/sms-gateway.git
cd sms-gateway && make dev
```

## First task

Pick a `good-first-issue` and ship a small change end to end.
